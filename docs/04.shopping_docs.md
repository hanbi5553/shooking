# 🛍️ 쇼핑몰 연동 프로젝트

## 프로젝트 개요
- **목표:** 기존 쇼핑몰 페이지와 외부 서비스/API를 연동하여 상품 정보를 자동으로 가져오고 관리할 수 있는 기능 구현  
- **주요 기능:**
  - 쇼핑몰 상품 목록 연동  
  - 상품 정보 자동 업데이트  
  - 재고 및 가격 동기화  
  - 테스트 환경에서 연동 기능 검증  


## 개발 환경
- **프레임워크:** React 17.0  
- **언어:** JavaScript (ES6)  
- **라이브러리:** Axios (API 호출), Recoil (상태 관리)  
- **스타일링:** CSS / Tailwind CSS  

## 설치 및 실행 방법
```bash
# 의존성 설치
npm install

# 로컬 서버 실행
npm start

# 테스트 실행
npm test
